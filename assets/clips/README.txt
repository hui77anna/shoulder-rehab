真人循环视频 — 放这里
========================

把每个动作的一小段真人视频（3–6 秒、静音、动作完整一遍）放进这个文件夹，
命名成「动作 id.mp4」，网页里对应卡片的示意小人就会自动被替换成你的真人视频，
静音 + 自动循环播放。不用改任何代码，丢文件进来刷新即可。

文件名（必须完全一致，全小写 .mp4）：
  foam.mp4          泡沫轴胸椎伸展
  wallangel.mp4     墙天使
  pushupplus.mp4    肩胛俯卧撑
  sleeper.mp4       侧躺内旋拉伸 (Sleeper Stretch)
  crossbody.mp4     体前交叉拉伸
  bandER.mp4        弹力带外旋
  facepull.mp4      面拉
  yraise.mp4        Y 字举
  traise.mp4        T 字举
  wraise.mp4        W 字收
  er90.mp4          90° 外旋
  farmer.mp4        农夫行走
  landmine.mp4      半跪推举
  medball.mp4       过头抛球
  shadowserve.mp4   空拍发球

视频要求 / 建议：
  - 只能用「你自己拍的」或「CC0 / 免费可商用」素材。不要剪别人的 B站/油管视频（侵权）。
  - 竖屏或方形都行；时长 3–6 秒、把动作做完整一遍最适合循环。
  - 压到 1–3MB 一条最好（手机加载快）。可以用任意工具导出 720p、码率调低。
  - 没有某个动作的视频也没关系 —— 那张卡片就继续显示示意小人，不影响其它。

部署提醒：
  如果你把网页挂成链接（tiiny / Netlify / EdgeOne），记得把整个 assets 文件夹
  和 index.html 一起上传（打包成 zip 再传），视频才跟得过去。
